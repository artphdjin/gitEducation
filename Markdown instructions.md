# Инструкция для работы с Markdown

## Выделение текста

*Курсив* - _Курсив_
**Полужирный** - __Полужирный__

Альтернативные способы форматирования позволяют комбинировать разные способы форматирования.

## Списки

Ненумерованные списки * или +

* Первый элемент
* Второй элемент
* Третий элемент
+ Четвертый элемент

Ненумерованные списки

1. Элемент 1
2. Элемент 2
3. Элемент 3
